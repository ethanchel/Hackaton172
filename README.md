# Flight Tracker

A simple web app to track flights in real time. Enter a flight number, see its status, and save flights to your favorites.

## Tech Stack

- HTML / CSS / JavaScript
- Public aviation API (AviationStack or OpenSky)
- LocalStorage for data persistence

## Features

- Search a flight by number (e.g., AF1234)
- View flight status: on time, delayed, in flight, landed
- Display aircraft type, departure/arrival airports, and times
- Save flights to favorites
- Manage favorites list (view and remove)

## How to Run

1. Clone the repo
```bash
   git clone https://github.com/ethanchel/Hackaton172.git
```
2. Open `index.html` in your browser

No dependencies required.
